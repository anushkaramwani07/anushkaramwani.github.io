Difference between Absolute and Relative CSS length units.

Absolute length units are fixed and don't change no matter what.
    Example:
  • px
  • cm
  • mm
Relative Units on the other hand is flexible and it changes depending on the context.
    Example:
  • em: based on font size
  • %: also based on font size
  • vw
Units Description
px has fixed size and is the smallest unit. It is an absolute unit in CSS. 1 px represents a single dot on the screen.
pt(points) Mostly used in print or PDF styling. 1pt equals 1.33 px.
em it is equal to computed font size. Uses the font size of whatever the "font size" is set to. Example if the font--size is set to 16px, so 1em that will be 16px.
rem Size relative to the text size of the root of the page.
vw it is 1% of the screen's width. 1vw equals to 1% of the total screen width. Usefull for creating layout.
vh It is 1% of the screen's height. 1vh equals to 1% of the total screen height.
