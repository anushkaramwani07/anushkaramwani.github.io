Difference between Absolute and Relative CSS length units.

Absolute length units are fixed and don't change no matter what.
    Example:
  • px
  • cm
  • mm
Relative Units on the other hand is flexible and it changes depending on the context.
    Example:
  • em: based on font size
  • %: also based on font size
  • vw

px: has fixed size and is the smallest unit

pt: has fixed size and is the smallest unit
